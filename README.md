## 📌 Overview

**MyMirath** is a frontend Islamic inheritance calculator built with React, JavaScript, and Tailwind CSS. It simplifies complex Shariah rules by translating them into structured logic and presenting results through a clean, user-friendly interface.

---

## 🌟 Features

- Accurate Islamic inheritance calculations for all possible heirs  
- Multi-language support: **Pashto, Persian, Urdu, English, Arabic**  
- Clean, responsive, and intuitive UI built with Tailwind CSS  
- Modular and scalable React component architecture  
- Step-by-step display of inheritance shares
